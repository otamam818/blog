# Blog
A blog I am making by using GitHub as a backend via fetch requests

It uses fetch requests to get the `.md` file containing the blog post and WASM to parse it into a custom HTML element.

# Frameworks
## Rust
- handleBars-rs
## JavaScript
- Svelte
